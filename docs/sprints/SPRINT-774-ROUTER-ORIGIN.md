# Sprint 774 — Router Origin Fix

**Date:** 2026-03-12
**Theme:** Fix Expo Router origin in app.json to topranker.io
**Story Points:** 1 (hardening)

---

## Mission Alignment

- **Constitution #15:** One source of truth — router origin must match live domain
- **TestFlight readiness:** Expo Router uses origin for URL generation in the native app

---

## Problem

`app.json` had `expo-router` plugin origin and `extra.router.origin` both set to `https://topranker.com`. This affects how Expo Router generates URLs in the native app. Since `.com` DNS is not configured, any URL generated by the router would be broken.

## Fix

Updated both values to `https://topranker.io`. Deep linking `associatedDomains` and `intentFilters` still include both `.com` and `.io` for future-proofing.
