## What Could Improve

- **No A/B testing capability** — we can't measure whether the polish actually improves onboarding completion rate until we have analytics in place (Sprint 714).
- **Slide animations re-trigger** — FadeInDown/FadeInUp re-animate when navigating back to a previously seen slide. Not a bug but slightly jarring. Consider converting to function components with layout animations in a future sprint.
